# Node Hello World

Simple node.js app that servers "hello world"

Great for testing simple deployments to the cloud

## Run It

`npm start`

## Build docker image

`docker build -t nodejsapp .`

## Docker images check

`docker images`

## docker run

`docker run -it -p 3000:3000 nodejsapp`

Access http;//localhost:3000 -> hello world webpage
